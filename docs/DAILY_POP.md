# Daily Pop Screen

## Purpose

Daily Pop is a categorized report of supplied signals with material changes. It is not a generic top-ten list.

## Categories

1. Best New Buy Setups
2. Best Pullback Opportunities
3. Breakout Candidates
4. High-Quality Watchlist Names Near Buy Zone
5. Insider Conviction Pops
6. Danger / Risk-Gate Changes
7. Market Regime Shift
8. Lesson of the Day

## Rules

- Requires material change.
- Includes Why Now and Why Not.
- Uses Signal Review Board for actionable categories.
- Danger items may appear despite failing Buy Setup review.
- Excludes stale already-good names without a material change.
- Respects max-name preferences.

## Limitations

No universe fetching, external data, real levels, or journal integration is implemented in Phase 2.
